### 什么是正向工程？什么是逆向工程？
正向工程 就是将数据库设计软件（例如 ERWin）画好的数据表图通过设计软件在数据库中生成最终的数据表，例如从CDM到PDM，PDM到Database等。逆向工程是把数据库中的数据表映射到设计软件中，以图表显示，例如Database  --> PDM --> CDM。
